# 方案待选：激进特征重构与算法改进

> **日期**: 2026-03-24
> **目标**: R² > 0.93 (Nested CV), 冲击 0.95
> **资源**: 2× A800 GPU, 不限计算时间
> **核心策略**: 从 "46 维手工特征 + TabPFN" 升级为 "多源信息融合 + 大规模预训练 + 物理约束集成"

---

## 零、现实检查

### 为什么 R²=0.95 很难？

| 障碍 | 原因 | 能否克服？ |
|------|------|-----------|
| 分子量未知 | 数据集不提供 MW, Fox-Flory 效应无法建模 | ❌ 数据限制 |
| 立构规整性 | SMILES 不含 i/a/s 信息, ΔTg ≈ 10-30K | ❌ 表示限制 |
| 测量噪声 | DSC/DMA/介电谱给出不同 Tg, ΔTg ≈ 5-15K | 部分可以（数据清洗） |
| 热历史/加工条件 | 退火/冷却速率, ΔTg ≈ 3-5K | ❌ 数据限制 |
| 多源数据不一致 | 4 个来源的测量标准不统一 | ✅ 可清洗 |

**理论上限估计**: 如果消除测量噪声和数据不一致，R² 天花板可能从 0.90 提升到 **0.92-0.94**。要到 0.95 需要**接近完美的数据 + 最强的模型**。

### 当前最优 vs 目标

```
当前: TabPFN R² = 0.8955 (Nested CV, 7486 条)
目标: R² > 0.93

差距: ~3.5% R² ≈ 需要解释额外 35% 的残余方差
当前残余方差 = (1 - 0.8955) × Var(Tg) ≈ 10.45% × Var(Tg)
目标残余方差 = (1 - 0.93) × Var(Tg) ≈ 7% × Var(Tg)
需减少: 10.45% → 7% ≈ 减少 33% 的残余误差
```

## 一、信息来源升级 — 5 个新信息维度

当前 M2M-V 只使用了**一个信息来源**：单体 2D SMILES → 手工特征。要大幅提升，必须引入新的信息维度。

### 维度 1：GNN 预训练嵌入 (64-128 dim)

**原理**: 在 22K+ 聚合物数据上预训练 GNN，学习"聚合物图 → 性质"的通用映射，然后提取中间层嵌入作为特征。

**操作**:
```
Step 1: 汇聚预训练数据
  - unified_tg.parquet: 7,486 条 (Tg 标签)
  - neurips_opp_tg.csv: 7,367 条 (Tg 标签)
  - polymetrix_tg.csv: 7,367 条 (Tg 标签)
  - 去重后预计: ~15,000-18,000 条
  - Fox 共聚物: ~46,000 条 (合成数据, 可选)

Step 2: 预训练 PhysicsGAT (A800)
  - 架构: 3 层 GAT + GRIN 池化
  - 目标: 预测 Tg (回归)
  - epochs: 200, batch=256, lr=1e-3
  - 预训练数据: ~18K 去重聚合物
  - 预计训练时间: ~2-4 小时 (A800)

Step 3: 提取嵌入
  - 冻结前 2 层 GAT
  - 第 3 层输出: 64-dim 图级嵌入
  - 对 7,486 统一数据集提取嵌入

Step 4: 融合
  - X_new = concat(M2M-V 46d, GNN_embed 64d) = 110d
  - 输入 TabPFN / CatBoost
```

**已有代码**: `src/gnn/pretrainer.py` (326 行), `src/gnn/physics_gat.py` (220 行), `src/gnn/graph_builder.py`
**预期提升**: R² +1.0-2.5% (GNN 嵌入捕捉了拓扑模式，与手工特征互补)

---

### 维度 2：polyBERT / 预训练语言模型嵌入 (768 dim)

**原理**: polyBERT 在 1 亿+ 聚合物 PSMILES 上预训练，学习了化学语言的深层模式。其 768 维嵌入编码了手工特征无法触及的结构信息。

**操作**:
```
Step 1: 安装 polyBERT
  pip install polyBERT  # 或从 GitHub 克隆

Step 2: 提取嵌入
  from polybert import PolyBERT
  model = PolyBERT.from_pretrained()
  embeddings = model.encode(smiles_list)  # [7486, 768]

Step 3: 降维 (可选)
  - PCA 768→64 或 128 (减少共线性和计算量)
  - 或直接用全 768d (TabPFN 可以处理)

Step 4: 融合
  X_new = concat(M2M-V 46d, polyBERT 768d) = 814d
  或: concat(M2M-V 46d, polyBERT_PCA 64d) = 110d
```

**NeurIPS 2025 Kaggle 冠军已验证**: polyBERT 嵌入 + AutoGluon 在竞赛中获胜
**预期提升**: R² +1.0-3.0% (大规模预训练的知识迁移)

---

### 维度 3：3D 构象描述符 (CES, 5-10 dim)

**原理**: 学习计划第四阶段提出的构象熵描述符，利用 A800 加速 3D 构象采样。

**操作**:
```
Step 1: 对 7,486 分子做大规模构象采样
  - 每分子 200 个构象 (A800 加速, 用 GPU 版 RDKit 或 OpenMM)
  - MMFF/UFF 优化
  - 预计: ~1-2 小时 (A800 并行)

Step 2: 计算构象描述符 (扩展到 10 维)
  CES_diversity           ← RMSD 分散度
  CES_energy_spread       ← 构象能量方差
  CES_gyration_var        ← 回转半径变异
  CES_volume_ratio        ← vdW/包络体积比
  CES_max_dihedral_range  ← 二面角范围
  CES_asphericity         ← 构象非球形度
  CES_eccentricity        ← 离心率
  CES_pmi_ratio           ← 主惯量矩比
  CES_sasa_var            ← 溶剂可及面积变异
  CES_strain_energy       ← 最低能与平均能的差
```

**预期提升**: R² +0.5-2.0% (全新的 3D 信息维度)

---

**原理**: 替换 FlexibilityIndex，按键类型加权旋转难度。

### 维度 5：Van Krevelen GC 预测值 (1-2 dim)

## 二、数据层面改进

### 2.1 数据清洗 — 移除高噪声样本

```
Step 1: 用 TabPFN 做 out-of-fold 预测，找出 |error| > 3σ 的异常点
Step 2: 人工审查这些异常点:
  - 是否是测量错误？(如 Tg 标注了 Tm)
  - 是否来自低质量数据源？
  - SMILES 是否正确？
Step 3: 移除确认的噪声数据 (预计 2-5%, ~150-375 条)
Step 4: 用清洗后数据重新训练

预期: R² +0.5-1.5% (噪声数据同时污染训练和评估)
```

### 2.2 SMILES 标准化与增强

```
Step 1: 对所有 SMILES 做规范化 (RDKit canonical)
Step 2: 验证聚合物标记 (* 或 [*]) 一致性
Step 3: 可选: SMILES 枚举增强 (多种等价表示 → 预训练用)
```

### 2.3 扩大预训练数据

```
当前: 7,486 条 (目标集)
可用: ~22K+ 条 (含 NeurIPS, PolyMetriX, 等外部数据)
可生成: ~46K Fox 共聚物 (合成数据, 可选用于预训练)
合计: ~70K 条用于 GNN/语言模型预训练
```

## 三、模型架构升级

### 3.1 多源信息融合 Stacking

```
信息源层 (5 种信息输入):
  ├─ M2M-V 46d (手工物理特征)
  ├─ GNN 嵌入 64-128d (图结构)
  ├─ polyBERT 64-128d (语言模型)
  ├─ CES 10d (3D 构象)
  └─ RBP 4d + GC 2d (物理精细化)

基模型层 (异质, 各自使用不同特征子集):
  ├─ TabPFN v2 (全量特征 200+ dim)           → ŷ₁
  ├─ CatBoost (物理特征 62d, 单调约束)        → ŷ₂
  ├─ LightGBM (全量特征 200+ dim)             → ŷ₃
  ├─ TandemM2M 端到端 (图 + 表格, GPU)       → ŷ₄
  ├─ Ridge (GC 特征 + polyBERT PCA 10d)      → ŷ₅
  └─ KNN (GNN 嵌入空间, k=15)               → ŷ₆

元学习器层:
  Ridge / ElasticNet / 小型 MLP
  输入: [ŷ₁, ŷ₂, ŷ₃, ŷ₄, ŷ₅, ŷ₆] + 核心物理特征 5d
  → 最终预测 Tg
```

### 3.2 预训练 → 微调 → 融合 三阶段流水线

```
Stage 1: 大规模预训练 (A800, ~70K 数据)
  ├─ GNN: PhysicsGAT 在 ~22K 聚合物上预训练 (GPU, ~4h)
  └─ polyBERT: 直接使用开源预训练权重 (无需重训)

Stage 2: 微调 + 嵌入提取 (A800, 7486 数据)
  ├─ GNN: 冻结前 2 层, 微调最后层 → 提取 64d 嵌入
  ├─ polyBERT: 提取 768d 嵌入 → PCA 128d
  └─ CES: 200 构象/分子, GPU 加速 (~2h)

Stage 3: 特征融合 + Stacking (CPU/GPU)
  ├─ 构建 200+ 维融合特征矩阵
  ├─ 6 基模型 Nested CV 训练
  ├─ 元学习器训练
  └─ 最终评估
```

### 3.3 物理约束注入

```python
# CatBoost 单调约束
monotone_constraints = {
    'RBP_weighted_flex': -1,       # 柔性↑ → Tg↓
    'ced_weighted_sum': 1,         # CED↑ → Tg↑
    'CES_diversity': -1,           # 构象多样性↑ → Tg↓
    'VPD_RingCount_per_RU': 1,    # 环密度↑ → Tg↑
    'Tg_GC': 1,                   # GC预测↑ → Tg↑
}

# 后处理物理边界
y_pred = np.clip(y_pred, 100, 700)
```

## 四、预期效果叠加分析

### 各改进的预期增量 (Nested CV R²)

| 改进 | 独立贡献 | 叠加后累计 | 理由 |
|------|---------|-----------|------|
| **基线 (TabPFN + M2M-V)** | — | **0.8955** | 当前最优 |
| + 数据清洗 (去噪声) | +0.5-1.5% | **0.900-0.910** | 减少不可约噪声 |
| + RBP (旋转势垒) | +0.3-0.5% | **0.903-0.915** | 柔性精细化 |
| + GC 特征 | +0.3-0.5% | **0.906-0.920** | 物理先验 |
| + GNN 预训练嵌入 | +1.0-2.0% | **0.916-0.935** | 图结构新信息 |
| + polyBERT 嵌入 | +0.5-1.5% | **0.920-0.940** | 语言模型知识 |
| + CES 构象熵 | +0.3-1.0% | **0.923-0.945** | 3D 信息 |
| + 异质 Stacking | +0.5-1.0% | **0.928-0.950** | 模型多样性 |
| + 物理约束 | +0.1-0.3% | **0.929-0.953** | 外推一致性 |

> **注意**: 各改进不完全独立，实际叠加效果可能低于简单加和。保守估计最终 R² 在 **0.92-0.94** 区间，乐观估计可触及 **0.95**。

### 关键不确定因素

| 因素 | 影响 | 如何应对 |
|------|------|---------|
| GNN 嵌入与手工特征的信息冗余度 | 冗余高→增量小 | 消融实验验证 |
| polyBERT 嵌入的 Tg 相关性 | 通用嵌入可能不专注 Tg | PCA + SHAP 筛选 |
| 数据清洗的激进程度 | 去太多→样本不足 | 只去 3σ 以上异常 |
| Nested CV 的严格性 | 越严格 R² 越低 | 保持严格，这是我们的竞争优势 |

## 六、最坏/最好/最可能结果

| 情景 | 最终 R² (Nested CV) | 条件 |
|------|---------------------|------|
| **最坏** | 0.91-0.92 | GNN/polyBERT 嵌入与手工特征高度冗余 |
| **最可能** | 0.93-0.94 | 多源融合有效，数据清洗去除噪声 |
| **最好** | 0.94-0.96 | 所有改进叠加有效 + 数据质量好 |

**R²=0.95 的必要条件**:
1. 数据集的固有测量噪声 < 10K MAE
2. GNN 嵌入和 polyBERT 嵌入与手工特征的信息冗余度 < 50%
3. 数据清洗有效移除了 3-5% 的噪声样本
4. 异质 Stacking 的模型多样性足够高

**即使最终 R² 没有达到 0.95，这个方案本身就是论文的重大创新**：
- "多源信息融合（手工物理特征 + GNN 嵌入 + 语言模型嵌入 + 3D 构象）用于聚合物 Tg 预测" — 文献空白
- 每个信息源的消融分析有方法论价值
- 物理约束 + 数据驱动的融合是新范式
