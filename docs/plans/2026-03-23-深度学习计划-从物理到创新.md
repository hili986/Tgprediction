## 第一阶段：Tg 的物理本质（第 1-2 周）

### 目标
从分子层面真正理解：**什么决定了一个聚合物的 Tg？为什么？**

### 1.1 核心问题链（带着问题读）

读任何材料之前，先把这些问题写在纸上，读完后必须能回答：

1. **Tg 的微观本质是什么？**
   - 玻璃化转变在分子层面发生了什么？
   - 为什么叫"转变"而不是"相变"？和一级相变（Tm）的本质区别？

2. **三大理论各说了什么？它们矛盾吗？**
   - 自由体积理论：Tg 发生在 f = f_g ≈ 0.025 时，为什么这个阈值是常数？
   - WLF/动力学理论：τ(T) 的超 Arrhenius 行为意味着什么？
   - Gibbs-DiMarzio 热力学理论：构象熵 S_c → 0 意味着什么？
   - **关键**：这三个理论本质上是不是在说同一件事的不同方面？

3. **什么结构因素控制链段运动？**
   - 主链刚性（旋转势垒）→ 具体怎么影响？举三个例子
   - 侧基效应 → 大侧基升高 Tg（阻碍运动）还是降低 Tg（增加自由体积）？答案是"看情况"——看什么？
   - 分子间相互作用 → 氢键、π-π 堆积、偶极-偶极，各自对 Tg 的影响量级？
   - 交联密度 → 为什么轻度交联升高 Tg 但重度交联后 Tg 概念本身就模糊了？

4. **从单体 SMILES 到聚合物 Tg，中间丢失了什么信息？**
   - 列出至少 5 个"单体 SMILES 无法编码"的物理效应
   - 这些效应对 Tg 的贡献量有多大？（这就是天花板 R²≈0.84 的来源）

## 第四阶段：提出原创想法（第 4-6 周）

### 目标
基于前三个阶段的积累，**提出 2-3 个原创的特征或方法假说**，设计验证实验。

### 4.1 创新思考框架

从三个维度寻找创新空间：

#### 维度 A：物理信息的更深度编码

目前 VPD 只模拟了 3-mer 线性链。还有什么物理信息可以编码？

思考方向：
- **构象空间采样**：不只是模拟一个构象，而是用 RDKit 的 conformer generator 采样多个 3D 构象，计算**构象多样性**（高多样性 → 高柔性 → 低 Tg）
- **链段体积效率**：用 3D 构象估算单体在空间中的**填充效率**——这直接对应自由体积理论
- **旋转势垒估算**：从主链二面角的能量曲线估算旋转势垒——这是链柔性的物理根源
- **分子间作用力强度**：超越"氢键数量"，估算**氢键强度、π-π 堆积能量、偶极矩大小**

#### 维度 B：表示方法的创新

思考方向：
- **多尺度表示**：atom → functional group → monomer → dimer/trimer → polymer，每个尺度提取不同信息
- **物理引导的图表示**：不用 SMILES 的分子图，而是构建"物理交互图"（节点是功能基团，边是物理相互作用）
- **混合表示**：传统描述符 + GNN 嵌入 + 物理约束，三者融合

#### 维度 C：利用现有数据的新方式

思考方向：
- **残差分析**：现有模型预测错误最大的分子有什么共同特点？→ 这些特点对应什么物理效应没被捕捉？
- **物理一致性检验**：模型预测是否满足已知物理规律？（如 Tg < Tm、侧基越大 Tg 趋势等）
- **数据增强新思路**：不只是 Fox 混合，能否用 GC 基团贡献法生成更多合理的 Tg 数据？

### 4.2 创新假说模板

每个想法用这个模板记录：

```markdown
## 假说 [编号]：[名称]

### 物理依据
- 基于什么 Tg 理论？
- 这个物理量如何影响链段运动？

### 具体实现
- 怎么计算？用什么工具？
- 计算复杂度？

### 预期效果
- 预计能提升多少？为什么？
- 和现有哪些特征互补？哪些冗余？

### 验证实验
- 加入现有 pipeline，对比有/无这个特征的 R²
- SHAP 排名预期：应该在 Top N

### 失败条件
- 如果 R² 没有提升，说明什么？
- 有什么备选方案？
```

### 4.3 一些具体的种子想法（供启发，不要照搬）

以下想法不一定对，但可以作为思考的起点：

**种子 1：构象熵描述符**
- 物理依据：Gibbs-DiMarzio 理论——Tg 发生在构象熵 S_c → 0 时
- 实现：用 RDKit 生成 N 个 3D 构象，计算构象多样性指标
- 预期：直接编码 Tg 的热力学本质，应该是强预测因子

**种子 2：协同运动长度估算**
- 物理依据：Adam-Gibbs 理论——Tg 与"协同运动区域"大小有关
- 实现：从单体结构估算链段的"最小运动单元"包含几个原子
- 预期：这是文献空白——没有人从分子结构直接估算协同运动长度

**种子 3：分子间作用力谱**
- 物理依据：分子间相互作用是 Tg 的关键因素，但现有特征只计数氢键数量
- 实现：计算一个"相互作用指纹"——不只是氢键，还包括 π-π、偶极-偶极、范德华力的相对强度
- 预期：更全面地描述分子间作用力，应该比单纯的 HBondDonors/Acceptors 更有效

**种子 4：自由体积近似描述符**
- 物理依据：自由体积理论——Tg 发生在 f ≈ 0.025 时
- 实现：从 3D 构象估算单体的 van der Waals 体积 / 包络体积比
- 预期：直接编码自由体积理论的核心参数

### 4.4 验证标准

- [ ] 提出至少 2 个有物理依据的原创特征假说
- [ ] 每个假说有完整的"物理依据 → 实现方案 → 预期效果 → 实验设计"链条
