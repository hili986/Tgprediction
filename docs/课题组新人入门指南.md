## 一、我们要解决什么问题？

### 1.1 什么是玻璃化转变温度 (Tg)

先用生活中的例子来理解：

- 冬天室外的塑料管被踩一脚会"啪"地碎裂 → 它处于 **玻璃态**（低于 Tg）
- 橡皮筋在室温下柔软有弹性 → 天然橡胶的 Tg 约 -70°C，远低于室温，处于 **高弹态**

**Tg (Glass Transition Temperature)** 就是高分子从"硬而脆的玻璃"变成"柔软有弹性的橡胶"的临界温度。它是高分子材料**最重要的热力学性质之一**。

```
温度升高 →
┌──────────────┬──────────────────┬──────────────────┐
│  玻璃态       │  玻璃化转变区     │  高弹态           │
│  链段冻结     │  链段开始运动     │  链段自由运动     │
│  硬、脆       │  粘弹性          │  柔软、有弹性     │
└──────────────┴──────────────────┴──────────────────┘
                      ↑ Tg 在这里
```

常见聚合物的 Tg 示例：

| 聚合物 | 简称 | Tg (°C) | 室温下状态 | 日常用途 |
|--------|------|---------|-----------|---------|
| 天然橡胶 | NR | -70 | 弹性体 | 轮胎、手套 |
| 聚丙烯 | PP | -10 | 半刚性塑料 | 食品容器 |
| 聚苯乙烯 | PS | 100 | 刚性塑料 | 泡沫饭盒 |
| 聚碳酸酯 | PC | 150 | 透明刚性 | 眼镜片、光盘 |
| 聚酰亚胺 | PI | 360 | 耐高温薄膜 | 航天、电子 |

### 1.2 为什么要用 AI 预测 Tg？

传统方法测量 Tg 需要先**合成**样品（耗时天~周），再用 DSC/DMA 等仪器测量。如果能直接从分子结构预测 Tg，就能跳过合成步骤，在**虚拟筛选**阶段就淘汰不合适的候选材料。

**一句话总结我们的项目**：输入一个高分子的化学结构（SMILES），输出它的 Tg 预测值。

```
输入: O=C(O[CH2][CH2])    →    AI 模型    →    输出: Tg = 340K (67°C)
     （聚合物的 SMILES）                        （预测的玻璃化转变温度）
```

### 1.3 这个问题的难点

| 挑战 | 解释 |
|------|------|
| **数据稀缺** | 公开数据集仅 304 条均聚物（对比 CV 领域动辄百万级图片） |
| **理论无解析解** | Tg 是一个动力学现象，没有第一性原理公式可以直接算 |
| **结构多样性** | 从简单的聚乙烯到复杂的核酸，分子结构跨度巨大 |
| **迁移预测** | 从均聚物训练的模型，能预测核酸这类完全不同的分子吗？ |

## 二、项目的核心创新点

我们的研究不是简单地"跑一个模型"，而是围绕几个关键创新展开：

### 2.1 虚拟聚合描述符 (VPD) — 本项目最大突破

**问题**：传统方法只分析单体的分子特征，但聚合物的性质取决于**聚合后**的链结构。

**我们的方案**：提出了 **VPD (Virtual Polymerization Descriptors)**，虚拟地将单体"聚合"成 3 个重复单元的寡聚体，再计算描述符，捕捉聚合带来的结构变化。

```
单体 SMILES → 虚拟聚合 → 3-RU 寡聚体 → 计算聚合前后的变化
                                          │
                                          ├── 环数变化 (RingCount_per_RU)
                                          ├── 柔性变化 (junction_flex_ratio)
                                          ├── 极性变化 (TPSA_per_RU)
                                          └── ... 共 12 维特征
```

**效果**：VPD 使预测精度提升了 4.6%（R² 从 0.82 → 0.866），是方案 A 最大的性能突破。

### 2.2 多层级特征体系 (M2M-V)

我们设计了一套 46 维的多层级特征：

| 层级 | 特征数 | 来源 | 描述 |
|------|--------|------|------|
| L0 (Afsordeh) | 4 | 文献复现 | 柔性指数、溶解度参数、氢键密度、极性指数 |
| L1 (RDKit) | 15 | 化学信息学 | 分子量、环数、sp3 碳比例、极性表面积等 |
| L1H (HBond) | 15 | 自主设计 | 氢键供体/受体、内聚能密度估计等 |
| L2 (VPD) | 12 | **本项目创新** | 虚拟聚合描述符 |

### 2.3 核酸 Tg 迁移预测 — 文献空白

目前**没有文献**从聚合物 Tg 数据迁移预测核酸的 Tg。我们通过引入 205 条"桥梁聚合物"数据（结构介于普通聚合物和核酸之间），实现了核苷酸 Tg 预测：

| 核苷酸 | 预测误差 (K) | 说明 |
|--------|------------|------|
| ATP | 4.7 | [优] 误差很小 |
| ADP | 0.4 | [优] 几乎精准 |
| AMP | 8.7 | [可] 可接受 |

### 3.2 关键实验结果

**Phase 4（小数据消融实验，304 条）**：

| 实验 | 特征 | 模型 | R² | 关键发现 |
|------|------|------|-----|---------|
| E3（最佳） | M2M-V 46d | GBR | **0.866** | VPD +4.6%，核心突破 |
| E1（基线） | L1H 34d | GBR | 0.820 | 无 VPD 基线 |
| E9-15 | GNN 端到端 | GAT | <0 | 304 条完全无法训练 GNN |

**Phase 5（大数据模型对比，7,486 条）**：

| 排名 | 模型 | CV R² | CV MAE(K) | 特点 |
|------|------|-------|----------|------|
| 1 | **TabPFN v2** | **0.8955** | 24.05 | 零调参即 SOTA |
| 2 | Stacking v2 | 0.8750 | 27.99 | 4 模型集成 |
| 3 | XGBoost | 0.8709 | 28.26 | 经典树模型 |
| 4 | CatBoost | 0.8691 | 29.01 | 可解释性最佳 |

**关键结论**：
- **SHAP Top 15 特征 = 全量 46 维的精度**（1/3 特征量，零精度损失）
- **不确定性量化**：90% 覆盖率的预测置信区间已实现
- **TabPFN v2 零调参击败所有调参模型**，验证了"小数据上基础模型 > 手动调参"

## 四、技术方案概览

### 4.1 整体流程

```
┌─────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│ 分子结构  │ →  │ 特征工程  │ →  │ 特征选择  │ →  │ 模型训练  │ →  │ Tg 预测  │
│ (SMILES) │    │ (46 维)  │    │ (Top 15) │    │ (TabPFN) │    │ + 置信区间│
└─────────┘    └──────────┘    └──────────┘    └──────────┘    └──────────┘
                    │
     ┌──────────────┼───────────────┐
     │              │               │
 ┌───┴───┐    ┌────┴────┐    ┌────┴────┐
 │ L0/L1  │    │  HBond  │    │  VPD    │
 │化学描述│    │ 氢键特征 │    │虚拟聚合 │
 │  19维  │    │  15维   │    │ 12维   │
 └───────┘    └─────────┘    └─────────┘
```

### 4.2 核心工具与依赖

| 工具 | 用途 |
|------|------|
| **RDKit** | 化学信息学库，计算分子描述符 |
| **scikit-learn** | 机器学习模型（GBR、ExtraTrees、Stacking） |
| **CatBoost/XGBoost/LightGBM** | 梯度提升树模型 |
| **TabPFN v2** | 基于 Transformer 的表格预测基础模型 |
| **SHAP** | 模型可解释性分析 |
| **MAPIE** | 不确定性量化（共形预测） |
| **PyTorch + PyG** | GNN 图神经网络（方案 B） |

### 4.3 评估方法

采用 **Nested Cross-Validation（嵌套交叉验证）**，避免信息泄露：
- 外层：RepeatedKFold(n_splits=5, n_repeats=3) — 评估泛化能力
- 内层：KFold(3) — 超参数优化

这确保了我们报告的 R² 是对**未见数据**的真实预测能力，不是过拟合的结果。
