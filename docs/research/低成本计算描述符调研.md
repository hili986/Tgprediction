## 1. 基团贡献法

### 1.1 概述

基团贡献法 (Group Contribution, GC) 是最经典的低成本 Tg 预测方法，通过将聚合物重复单元分解为标准化学基团，利用各基团的加和贡献计算物性。

### 1.2 主要方法

#### Van Krevelen 方法
- **原理**：定义摩尔 Tg 函数 Yg，Tg = 1000 * Sum(Yg_i) / M（M 为重复单元摩尔质量）
- **基团数**：约 40 个标准基团
- **精度**：传统方法 R2 约 0.85-0.90
- **优势**：简单直观，手工可算
- **局限**：不考虑基团间相互作用、位置效应

#### Bicerano 方法
- **原理**：基于连接性指数（connectivity indices）和结构参数
- **核心参数**：
  - N_BB：主链原子数
  - N_SG：侧基大小参数
  - sigma：对称数
  - 零阶/一阶连接性指数
- **精度**：R2 约 0.90-0.93
- **优势**：考虑了拓扑连接性，更好捕获链刚性
- **参考书**：Bicerano, "Prediction of Polymer Properties" (3rd Ed., 2002)

#### Askadskii-Matveev 方法
- **原理**：基于非谐振子模型（anharmonic oscillator），考虑原子增量和分子内相互作用
- **特点**：比 Van Krevelen 多考虑了分子内能的非谐效应
- **精度**：与 Bicerano 相当

#### 改进型基团贡献法 (Satyanarayana et al., 2020)
- **创新**：将基团数扩展到 58 个，包含更多次级基团和位置异构体
- **精度**：R2 = 0.9925（在 450+ 种聚合物上验证）
- **关键改进**：区分了邻位/间位/对位取代，增加了杂原子基团
- **论文**：Satyanarayana et al., "Polymer Tg prediction by modified group contribution method" (2020)

### 1.3 计算成本

| 方法 | 所需输入 | 计算时间 | 是否需要软件 |
|------|----------|----------|-------------|
| Van Krevelen | 重复单元结构 | 手工 ~5 min | 否（查表） |
| Bicerano | SMILES/结构 | < 1 sec | Python 脚本 |
| 改进 GC | SMILES/结构 | < 1 sec | Python 脚本 |

### 1.4 中间变量

GC 法不仅预测 Tg，还可计算以下中间变量（均与 Tg 相关）：
- **内聚能密度 (CED)**：Ecoh/V，反映分子间作用力强度
- **摩尔体积 (Vm)**：反映分子堆积效率
- **链刚性参数**：主链转动自由度
- **氢键密度**：基团中氢键供/受体数量

### 1.5 与 Tg 的相关性

| 方法 | 数据集大小 | R2 | MAE (K) | 来源 |
|------|-----------|-----|---------|------|
| Van Krevelen 经典 | ~200 | 0.85-0.90 | 30-50 | Van Krevelen (1990) |
| Bicerano | ~320 | 0.90-0.93 | 20-35 | Bicerano (2002) |
| 改进 GC (2020) | 450+ | 0.9925 | ~10 | Satyanarayana (2020) |

## 2. 量子化学/DFT 单体描述符

### 2.1 概述

对聚合物单体或短寡聚物进行量子化学计算（DFT 或半经验方法），提取电子结构和几何信息作为描述符。关键优势是能捕获 GC 法忽略的电子效应。

### 2.2 主要描述符

#### 转动势垒 (Rotational Barriers)
- **定义**：主链和侧链关键二面角的旋转能垒
- **与 Tg 关系**：**强正相关**——势垒越高，链越刚性，Tg 越高
- **计算方法**：固定二面角做刚性/柔性扫描，B3LYP/6-31G* 水平
- **计算成本**：单个单体 ~10-30 min（DFT），~1-5 min（xTB）
- **证据**：Mattioni & Jurs (2002) 发现转动势垒是 Tg 预测中最重要的描述符之一

#### 偶极矩 (Dipole Moment)
- **定义**：分子电荷分布的极性度量
- **与 Tg 关系**：正相关（偶极-偶极相互作用增强链间作用力）
- **计算**：单点 DFT 或 xTB，< 1 min
- **相关系数**：单独 r ~ 0.3-0.5，组合后贡献显著

#### 极化率 (Polarizability)
- **定义**：分子在外电场下电子云变形的容易程度
- **与 Tg 关系**：复杂，与 van der Waals 相互作用相关
- **计算**：需要 frequency 计算或有限场方法，~5-15 min（DFT）

#### HOMO-LUMO 能隙
- **定义**：前线轨道能量差
- **与 Tg 关系**：间接相关，反映化学稳定性和电子离域程度
- **计算**：单点 DFT，< 5 min
- **注意**：Kohn-Sham DFT 的 HOMO-LUMO gap 系统性偏小

#### 分子体积与表面积
- **定义**：Van der Waals 体积/表面积，溶剂可及表面积
- **与 Tg 关系**：通过自由体积和堆积效率间接影响 Tg
- **计算**：几何优化后直接得到，< 1 min

#### 部分电荷分布
- **定义**：Mulliken、NBO 或 ESP 拟合电荷
- **与 Tg 关系**：反映局部极性和氢键能力
- **计算**：单点计算副产品，无额外成本

### 2.3 半经验替代方案：xTB (GFN2-xTB)

**强烈推荐作为 DFT 的低成本替代**：
- **速度**：比 DFT 快 100-1000 倍
- **精度**：几何结构和相对能量接近 DFT（误差 ~2-5 kcal/mol）
- **可算**：偶极矩、极化率、部分电荷、转动势垒、振动频率
- **软件**：xtb 程序（开源，Grimme 组开发）
- **适用规模**：可处理 500+ 原子体系（寡聚物片段）

### 2.4 计算成本汇总

| 描述符 | DFT (B3LYP/6-31G*) | xTB (GFN2) | 单独 r(Tg) |
|--------|--------------------|-----------|----|
| 转动势垒 | 10-30 min | 1-5 min | 0.5-0.7 |
| 偶极矩 | 1-5 min | < 30 sec | 0.3-0.5 |
| 极化率 | 5-15 min | < 1 min | 0.3-0.4 |
| HOMO-LUMO | 1-5 min | < 30 sec | 0.2-0.3 |
| 分子体积 | 1-5 min | < 30 sec | 0.4-0.6 |
| 部分电荷 | 1-5 min | < 30 sec | 0.3-0.5 |
| 全套描述符 | 30-60 min | 5-10 min | 组合 R2 ~0.85 |

## 3. 拓扑/图论描述符

### 3.1 概述

拓扑描述符从分子图（原子=节点，键=边）计算，不需要 3D 几何信息，计算成本极低（毫秒级）。

### 3.2 主要描述符

#### Wiener 指数 (W)
- **定义**：分子图中所有原子对之间最短路径长度之和
- **物理意义**：反映分子的紧凑程度和分支度
- **与 Tg 关系**：弱到中等相关（r ~ 0.3-0.5），因为同一 W 值可对应不同结构
- **计算**：O(N^2) 图算法，毫秒级

#### Randic 连接性指数 (chi)
- **定义**：chi = Sum over edges (delta_i * delta_j)^(-0.5)，delta 为原子度数
- **变体**：零阶 (0chi)、一阶 (1chi)、二阶 (2chi) 等
- **与 Tg 关系**：中等相关（r ~ 0.4-0.6），Bicerano 方法的核心输入
- **特点**：高阶连接性指数能捕获链分支和环结构信息

#### Balaban J 指数
- **定义**：基于距离矩阵的拓扑指数，对分子大小归一化
- **与 Tg 关系**：弱相关，更适合区分异构体
- **计算**：毫秒级

#### Kappa 形状指数
- **定义**：Hall-Kier kappa 指数（1kappa, 2kappa, 3kappa），比较分子与极端拓扑（线性 vs 完全分支）的差异
- **与 Tg 关系**：捕获分子形状对 Tg 的影响
- **计算**：毫秒级

#### 电拓扑状态指数 (E-state)
- **定义**：结合了拓扑环境和电负性的原子级描述符
- **与 Tg 关系**：比纯拓扑指数更好，因为包含了电子信息
- **计算**：毫秒级

### 3.3 拓扑描述符用于 Tg 预测的文献

| 研究 | 描述符组合 | 模型 | R2 | 数据集 |
|------|-----------|------|-----|-------|
| Katritzky et al. (1996) | 拓扑 + 几何 + 电子 | MLR | 0.90 | 88 聚合物 |
| Mattioni & Jurs (2002) | 拓扑 + QM | MLR/PLS | 0.93 | 165 聚合物 |
| Liu & Briber (2020) | 拓扑 + 宪法性 | SVR | 0.937 | 536 聚合物 |
| Yu et al. (2006) | E-state + 连接性 | SVM | 0.91 | 107 聚合物 |

### 3.4 局限性

- **单独使用 R2 较低**：纯拓扑描述符通常 R2 < 0.80
- **对立体化学不敏感**：无法区分顺/反异构、手性中心
- **对电子效应不敏感**：不区分 -OH 和 -F 的极性差异
- **最佳实践**：与其他描述符（电子、几何）组合使用


## 4. 化学信息学 2D 描述符

### 4.1 概述

化学信息学描述符是从 2D 分子结构（SMILES 或连接表）系统计算的数值特征，涵盖宪法性、拓扑、电子、几何等多个维度。数量大（可达 1000+），特别适合机器学习。

### 4.2 描述符库

#### RDKit 描述符 (~200 个)
- **类型**：
  - 宪法性：分子量、原子计数、键计数、环计数
  - 拓扑：Wiener 指数、BertzCT 复杂度指数
  - 电子：TPSA（拓扑极性表面积）、MR（摩尔折射率）、LogP
  - 氢键：HBD（供体数）、HBA（受体数）
  - 片段：BRICS 片段、函数基团计数
- **计算**：`from rdkit.Chem import Descriptors`，整套 < 1 sec
- **优势**：开源、成熟、广泛使用

#### Mordred 描述符 (1826 个，其中 1613 个 2D)
- **类型**：比 RDKit 更全面
  - 自相关描述符、BCUT 描述符
  - 信息内容描述符、分子框架描述符
  - 药效团描述符等
- **计算**：`from mordred import Calculator, descriptors`，整套 ~1-5 sec
- **优势**：描述符数量多，覆盖面广

#### Morgan/ECFP 指纹
- **类型**：圆形指纹，编码原子的局部化学环境
- **变体**：
  - ECFP4（半径 2，2048 位）：最常用
  - ECFP6（半径 3）：捕获更大范围的结构信息
  - FCFP（功能类指纹）：按药效团分组
- **计算**：`AllChem.GetMorganFingerprintAsBitVect(mol, 2, nBits=2048)`，< 1 sec
- **用于 Tg**：Pilania et al. (2019) 使用 Morgan FP + GPR 达到 R2 = 0.92

### 4.3 聚合物专用描述符

#### Polymer Genome 指纹
- **三级层次结构**：
  - Level 1（原子级）：原子类型和局部环境
  - Level 2（块级）：功能基团和环结构
  - Level 3（链级）：链骨架特征
- **模型**：GPR（高斯过程回归）
- **性能**：Tg 预测 R2 ~ 0.90-0.93
- **访问**：https://polymergenome.org

#### PolyMetriX (2025)
- **创新**：分层聚合物特征化器（hierarchical polymer featurizer）
- **特征分解**：
  - 全聚合物层 (Full polymer)：25 个化学描述符 + 7 个拓扑描述符
  - 骨架层 (Backbone)：同上，仅计算骨架部分
  - 侧链层 (Side chain)：同上，仅计算侧链部分
- **总描述符数**：(25+7) x 3 = 96 个
- **关键化学描述符**：
  - 分子量、原子计数（C/H/N/O/S/F/Cl 等）
  - 杂原子比例、不饱和度
  - 氢键供/受体数
  - 可旋转键数（chain flexibility 的直接度量）
  - 芳香环数和比例
  - sp3 碳比例
- **关键拓扑描述符**：
  - 链长度（原子数）
  - 分支度
  - 环化度
  - 路径复杂度
- **训练数据**：7367 个 Tg 数据点（来自多个数据库整合）
- **性能**：配合 Extra Trees/GPR 模型，R2 达 0.95-0.97
- **优势**：明确区分骨架/侧链贡献，可解释性强
- **论文**：PolyMetriX: A hierarchical polymer featurizer (2025)

### 4.4 性能对比

| 描述符方案 | 描述符数 | 模型 | R2 (Tg) | 数据集大小 |
|-----------|---------|------|---------|-----------|
| RDKit 基础 | ~200 | RF | 0.88-0.92 | ~500 |
| Mordred 2D | ~1600 | XGBoost | 0.90-0.94 | ~500 |
| Morgan FP (2048) | 2048 | GPR | 0.92 | ~500 |
| Polymer Genome | ~300 | GPR | 0.90-0.93 | ~800 |
| PolyMetriX | 96 | ET/GPR | 0.95-0.97 | 7367 |

## 5. 半经验力场参数

### 5.1 概述

从力场参数估算分子间相互作用强度，这些参数可以从量子化学计算或经验规则获得，不需要运行 MD 模拟。

### 5.2 主要参数

#### Lennard-Jones 参数 (epsilon, sigma)
- **定义**：
  - epsilon：势阱深度，反映原子间吸引力强度
  - sigma：有效原子直径
- **与 Tg 关系**：epsilon 与 CED 相关，间接影响 Tg
- **获取方式**：
  - OPLS-AA 力场参数表（直接查表）
  - GAFF（General Amber Force Field）自动分配
  - TraPPE 力场（适合聚合物）
- **成本**：查表 ~0，自动分配 < 1 sec

#### 部分电荷
- **方法**：
  - Gasteiger 电荷：纯拓扑方法，RDKit 内置，< 1 sec
  - AM1-BCC：半经验 AM1 + 经验校正，~1-10 sec
  - RESP：DFT + ESP 拟合，~5-30 min
- **与 Tg 关系**：反映偶极-偶极和氢键作用，与 Tg 间接正相关
- **推荐**：AM1-BCC（精度/成本最佳平衡）

#### 键力常数和角力常数
- **定义**：键伸缩和角弯曲的力常数
- **与 Tg 关系**：力常数越大，链越刚性，Tg 越高
- **获取**：力场参数表或 Seminario 方法（从 Hessian 矩阵提取）

#### 二面角参数
- **定义**：主链扭转势能面的 Fourier 系数
- **与 Tg 关系**：**强相关**——等效于转动势垒
- **获取**：力场参数表或 QM 扭转扫描拟合

### 5.3 衍生描述符

从力场参数可以进一步计算：
- **内聚能密度 (CED)**：Sum(epsilon_ij) / V
- **Hildebrand 溶解度参数**：delta = sqrt(CED)
- **分子间氢键能**：从 LJ + Coulomb 项估算

### 5.4 实用工具

| 工具 | 功能 | 成本 |
|------|------|------|
| Open Babel | 力场分配、Gasteiger 电荷 | 免费 |
| AmberTools (antechamber) | GAFF 参数、AM1-BCC 电荷 | 免费 |
| LAMMPS data tools | 力场参数提取 | 免费 |
| LigParGen | OPLS-AA 在线分配 | 免费在线 |

## 6. 粗粒化参数

### 6.1 概述

粗粒化参数描述聚合物链的整体构象特征，可以通过理论方法（不需要 MD 模拟）估算。

### 6.2 主要参数

#### 特征比 (Characteristic Ratio, C_inf)
- **定义**：<R^2> / (n * l^2)，衡量链的局部刚性（R 为端到端距离，n 为键数，l 为键长）
- **与 Tg 关系**：**强正相关**——C_inf 越大，链越刚，Tg 越高
- **理论估算方法**：
  1. **转移矩阵法 (RIS model)**：Flory 的旋转异构态模型
     - 输入：键角、转动势能面（3-态模型的能量参数）
     - 精度：对简单聚合物 +-10-20%
     - 计算：矩阵乘法，< 1 sec
  2. **Bicerano GC 估算**：
     - C_inf = f(连接性指数, N_BB, 环刚性参数)
     - 精度：R2 ~ 0.85 与实验值
     - 计算：< 1 sec

#### Kuhn 长度 (b_K)
- **定义**：b_K = C_inf * l（l 为键长），等效自由连接链的段长
- **与 Tg 关系**：与 C_inf 等效，强正相关
- **计算**：从 C_inf 直接得到

#### 持久长度 (Persistence Length, l_p)
- **定义**：链切线方向相关函数衰减到 1/e 的距离
- **与 Tg 关系**：与 C_inf 相关（l_p = C_inf * l / 2），强正相关
- **理论估算**：与 C_inf 相同方法

#### 柔性参数
- **可旋转键比例**：可旋转键数 / 总键数（RDKit 可直接计算）
- **骨架柔性指数**：骨架中 sp3 原子比例
- **环刚性贡献**：骨架环原子比例
- **与 Tg 关系**：这些简单比率与 Tg 中等负相关（r ~ -0.4 到 -0.6）

### 6.3 经验公式

Kuhn 温度概念：
- **Boyer-Simha**: Tg ~ 2/3 * Tm（粗略经验关系）
- **Simha-Boyer**: Tg * delta_alpha = 常数 ~ 0.113（delta_alpha 为热膨胀系数差）
- **链刚性-Tg 关联**: Tg = A + B * C_inf（线性回归，R2 ~ 0.7-0.8）

### 6.4 计算成本

| 参数 | 方法 | 计算时间 | 精度 |
|------|------|----------|------|
| C_inf | RIS 转移矩阵 | < 1 sec | +-15% |
| C_inf | Bicerano GC | < 1 sec | R2~0.85 |
| 可旋转键比例 | RDKit | < 1 sec | 精确 |
| l_p | 从 C_inf 导出 | < 1 sec | 同 C_inf |


## 7. 溶解度参数

### 7.1 概述

溶解度参数反映分子间作用力的总强度，是 Tg 的重要影响因素。可以通过基团贡献法低成本计算。

### 7.2 Hildebrand 溶解度参数

- **定义**：delta = sqrt(Ecoh / V)，Ecoh 为内聚能，V 为摩尔体积
- **单位**：(J/cm^3)^(1/2) 或 (cal/cm^3)^(1/2)
- **与 Tg 关系**：
  - 中等正相关（r ~ 0.4-0.6）
  - 溶解度参数高 -> 分子间作用力强 -> Tg 高
  - 但关系非线性（氢键聚合物偏离较大）

### 7.3 Hansen 溶解度参数

- **三分量分解**：
  - delta_d：色散力贡献
  - delta_p：极性力贡献
  - delta_h：氢键贡献
  - 关系：delta^2 = delta_d^2 + delta_p^2 + delta_h^2
- **与 Tg 关系**：
  - delta_h（氢键分量）与 Tg 相关性最强（r ~ 0.5-0.7）
  - delta_d 和 delta_p 补充了色散和极性信息
  - 三分量组合比 Hildebrand 单参数更准确

### 7.4 计算方法

| 方法 | 精度 | 成本 | 工具 |
|------|------|------|------|
| Van Krevelen GC | +-10% | < 1 sec | 查表/Python |
| Hoftyzer-Van Krevelen | +-5-10% | < 1 sec | 查表/Python |
| Hoy GC | +-10% | < 1 sec | 查表/Python |
| Fedors GC | +-10% | < 1 sec | 查表 |
| COSMOtherm | +-5% | ~30 min | 商业软件 |
| MD 模拟 | +-2-5% | 数小时 | LAMMPS/GROMACS |

**推荐**：Van Krevelen 或 Hoftyzer-Van Krevelen GC 法（精度足够，成本极低）

### 7.5 HSPiP 软件
- **功能**：Hansen 溶解度参数计算和可视化
- **方法**：基于 Y-MB (Yamamoto-Molecular Breaking) 方法
- **精度**：比传统 GC 法更准确
- **成本**：商业软件（学术版 ~$100）
- **替代**：RDKit + GC 基团表（免费）

## 8. 自由体积参数

### 8.1 概述

自由体积理论认为，Tg 是聚合物自由体积降至临界值（约 2.5%）时的温度。自由体积参数可以从分子结构直接估算。

### 8.2 主要参数

#### 分数自由体积 (Fractional Free Volume, FFV)
- **定义**：FFV = (V - V_occ) / V，V 为摩尔体积，V_occ 为占有体积
- **Bondi 方法**：V_occ = 1.3 * V_w（V_w 为 Van der Waals 体积）
- **与 Tg 关系**：
  - Simha-Boyer 理论：在 Tg 处 FFV 约为 0.025（2.5%）
  - 更精确的经验关系：FFV(Tg) ~ 0.113 * alpha（alpha 为热膨胀系数差）
  - **但 FFV 本身与 Tg 不是简单相关**——FFV 是 Tg 处的状态，不是预测 Tg 的输入

#### Van der Waals 体积 (V_w)
- **计算方法**：
  - Bondi 原子增量法：加和各原子的 Van der Waals 体积（查表）
  - RDKit：`Descriptors.MolMR()` 计算摩尔折射率（相关量）
  - 3D 方法：Connolly 分子表面积/体积（需 3D 构象）
- **与 Tg 关系**：通过 FFV 间接影响
- **成本**：Bondi 法 < 1 sec，3D 方法 ~1-5 sec

#### 堆积效率 (Packing Coefficient, K_p)
- **定义**：K_p = V_w / V，占有体积与总体积之比
- **与 Tg 关系**：K_p 越大，堆积越紧密，Tg 越高
- **典型值**：聚合物 K_p ~ 0.60-0.70

### 8.3 自由体积参数的局限

- **核心问题**：FFV 是 Tg 的结果而非原因——知道 FFV(Tg) 需要先知道 Tg
- **实用价值**：
  - 从结构估算的 V_w 和 K_p 可以作为描述符输入 ML 模型
  - 但不能单独用于 Tg 预测
  - 结合其他描述符（链刚性、相互作用强度）更有价值

### 8.4 计算工具

| 参数 | 方法 | 工具 | 成本 |
|------|------|------|------|
| V_w (Bondi) | 原子增量法 | 查表 / Python | < 1 sec |
| V_w (3D) | Connolly 表面 | RDKit / PyMOL | 1-5 sec |
| V (摩尔体积) | GC 法 | Van Krevelen 表 | < 1 sec |
| K_p | V_w / V | 上述工具组合 | < 1 sec |

### 9.2 推荐组合策略

#### 策略 A：纯数据驱动（最低成本，R2 ~ 0.95-0.97）
- **描述符**：PolyMetriX 96 维 + RDKit 描述符选择子集
- **模型**：Extra Trees 或 GPR
- **输入**：仅需 SMILES
- **计算时间**：每个聚合物 < 1 sec
- **适用**：大规模筛选、虚拟库搜索

#### 策略 B：物理增强（中等成本，R2 ~ 0.96-0.98）
- **描述符**：策略 A + GC 中间变量（CED、链刚性）+ 溶解度参数 + C_inf 估算
- **模型**：GPR 或梯度提升树
- **计算时间**：每个聚合物 < 5 sec
- **适用**：需要物理可解释性的研究

#### 策略 C：高精度（较高成本，R2 ~ 0.97-0.99）
- **描述符**：策略 B + xTB 电子描述符（转动势垒、偶极矩、极化率）
- **模型**：GPR 或神经网络
- **计算时间**：每个聚合物 5-15 min
- **适用**：精确预测特定聚合物系列

### 9.3 关键发现

1. **基团贡献法的回归**：改进 GC 法（R2=0.99）在准确度上可以与复杂 ML 方法媲美，且完全可解释
2. **分层描述最重要**：PolyMetriX 的成功表明，区分骨架/侧链的描述符比单一水平的描述符更有效
3. **数据量比描述符精度更重要**：7367 条数据 + 简单描述符 > 500 条数据 + 复杂描述符
4. **转动势垒是最有信息量的 QM 描述符**：如果只算一个 DFT/xTB 描述符，选转动势垒
5. **指纹方法的意外有效性**：Morgan 指纹虽然不可解释，但性能出色（R2 ~ 0.92）
6. **聚合物级 vs 单体级**：聚合物级描述符（考虑聚合度效应）始终优于纯单体描述符

### A2. Morgan 指纹计算

```python
from rdkit import Chem
from rdkit.Chem import AllChem
import numpy as np

def compute_morgan_fp(smiles, radius=2, n_bits=2048):
    mol = Chem.MolFromSmiles(smiles)
    if mol is None:
        return None
    fp = AllChem.GetMorganFingerprintAsBitVect(mol, radius, nBits=n_bits)
    return np.array(fp)

fp = compute_morgan_fp("C(C)c1ccccc1")
print(f"Fingerprint shape: {fp.shape}, Non-zero bits: {fp.sum()}")
```

## 13. Mordred 推荐描述符子集

从 1826 个 Mordred 描述符中，基于文献和特征选择研究推荐的约 50 个核心描述符：

### 宪法性描述符 (Constitutional) — 8个

- `nAtom`, `nHeavyAtom`: 原子数
- `nBonds`, `nBondsD`, `nBondsT`: 键数
- `nRot`: 可旋转键数 (**最重要**)
- `nRing`, `nAromRing`: 环数

### 拓扑描述符 (Topological) — 10个

- `BalabanJ`: Balaban J 指数
- `BertzCT`: 拓扑复杂度
- `Kappa1`, `Kappa2`, `Kappa3`: Hall-Kier kappa 形状指数
- `WPath`, `WPol`: Wiener 路径数/极性数
- `Zagreb1`, `Zagreb2`: Zagreb 指数
- `AATS0dv`: 自相关描述符

### 电子/极性描述符 (Electronic) — 10个

- `TPSA`: 拓扑极性表面积
- `SLogP`: Wildman-Crippen LogP
- `SMR`: 摩尔折射率
- `EstateVSA` 系列: 电拓扑态-表面积组合
- `PEOE_VSA` 系列: 部分均衡电负性-表面积组合

### 氢键描述符 (H-bond) — 5个

- `nHBDon`: 氢键供体数
- `nHBAcc`: 氢键受体数
- `TopoPSA`: 拓扑极性表面积
- `SsOH`, `SsNH`: O-H / N-H E-state

## 14. Afsordeh 4-Feature 结构特征详解 + SHAP 共识排序

### 14.1 Afsordeh & Shirali (2025) 研究详解

**论文**: Prediction of Glass-Transition Temperatures of Polymers: A Model Based on Structural Features of Repeating Units
**发表**: Chinese Journal of Polymer Science (2025)
**数据集**: 112 种聚合物，含聚烯烃、聚酯、聚酰胺、聚碳酸酯、聚砜、聚醚等
**最佳模型**: Extra Trees Regression, R²=0.97, MAE≈7K

#### 四个核心特征详解

| 特征 | 定义 | 物理含义 | 计算方法 |
|------|------|---------|----------|
| **Flexibility (F)** | 可旋转键数 / 主链原子数 | 链柔性越高，链越软，Tg越低 | RDKit: NumRotatableBonds / nAtom |
| **Side-chain Occupancy Length (SOL)** | 侧链最长支链长度 | 侧链越长，空间位阻越大，Tg越低 | 图论最长路径算法 |
| **H-bond Density (HBD)** | 氢键供体/受体密度 | 氢键越强，链间作用越强，Tg越高 | RDKit: (NumHDonors+NumHAcceptors) / nHeavyAtom |
| **Polarity Index (PI)** | 极性原子比例 | 极性越高，分子间作用越强，Tg越高 | (nO+nN+nS) / nHeavyAtom |

#### SHAP 重要性排序

Flexibility > SOL > HBD > PI，其中 Flexibility 的 SHAP 值显著高于其他三个特征。

#### Boyer-Simha 经验关系

该模型的物理基础是 Boyer-Simha 关系：

```
Tg = Ecoh / (R * Nbb)
```


其中 Ecoh 为内聚能密度，Nbb 为主链骨架键数。这建立了柔性与 Tg 的正向关联——可旋转键越多（Flexibility 越高），链段运动越容易，Tg 越低。

### 14.2 SHAP 多研究共识排序（综合 6+ 研究）

| 排序 | 特征类别 | SHAP 重要性 | 代表性描述符 |
|------|----------|-----------|-------------|
| 1 | **链柔性/刚性** | ★★★★★ | `NumRotatableBonds`, Flexibility |
| 2 | **拓扑复杂度** | ★★★★ | `BertzCT`, `BalabanJ`, `RingCount` |
| 3 | **极性/氢键** | ★★★ | `TPSA`, HBD, HBA, Polarity Index |
| 4 | **芳香环** | ★★★ | `NumAromaticRings`, `nAromAtom` |
| 5 | **侧链效应** | ★★ | SOL, `FractionCSP3` |
| 6 | **分子量/大小** | ★★ | `MolWt`, `nHeavyAtom` |

**关键发现**: 链柔性/刚性在所有研究中一致排名第一，是 Tg 最强的单一预测因子。拓扑复杂度排名第二，反映了环结构和支化对链段运动的限制。极性和氢键排名第三，体现分子间相互作用对 Tg 的贡献。

from rdkit import Chem
from rdkit.Chem import Descriptors, rdMolDescriptors

        if mol is None:
            results.append({})
            continue
        desc = {

            'RingCount': Descriptors.RingCount(mol),

            'NumAromaticRings': rdMolDescriptors.CalcNumAromaticRings(mol),

def compute_mordred_descriptors(smiles_list, use_3d=False):
    """批量计算 Mordred 描述符"""

### 16.3 Afsordeh 4 特征计算

```python
from rdkit import Chem
from rdkit.Chem import Descriptors, rdmolops
import pandas as pd

def compute_afsordeh_features(smiles):
    """计算 Afsordeh & Shirali (2025) 的 4 个结构特征"""
    mol = Chem.MolFromSmiles(smiles)
    if mol is None:
        return None
    
    n_heavy = mol.GetNumHeavyAtoms()
    
    # Feature 1: Flexibility = 可旋转键数 / 重原子数
    n_rot = Descriptors.NumRotatableBonds(mol)
    flexibility = n_rot / n_heavy if n_heavy > 0 else 0
    
    # Feature 2: Side-chain Occupancy Length (SOL)
    # 近似：使用最长非环路径长度 / 总重原子数
    try:
        longest = max(len(p) for p in 
                      Chem.rdmolops.FindAllPathsOfLengthN(mol, 7, useBonds=False))
    except:
        longest = 0
    sol = longest / n_heavy if n_heavy > 0 else 0
    
    # Feature 3: H-bond Density
    n_hbd = Descriptors.NumHDonors(mol)
    n_hba = Descriptors.NumHAcceptors(mol)
    hbond_density = (n_hbd + n_hba) / n_heavy if n_heavy > 0 else 0
    
    # Feature 4: Polarity Index
    polar_atoms = sum(1 for atom in mol.GetAtoms()
                      if atom.GetAtomicNum() in [7, 8, 9, 15, 16, 17, 35])
    polarity_index = polar_atoms / n_heavy if n_heavy > 0 else 0
    
    return {
        'Flexibility': flexibility,
        'SOL': sol,
        'HBondDensity': hbond_density,
        'PolarityIndex': polarity_index,
    }

def batch_afsordeh(smiles_list):
    results = [compute_afsordeh_features(smi) for smi in smiles_list]
    return pd.DataFrame([r for r in results if r is not None])
```

    '-COO-':    {'Ecoh': 18000, 'Vw': 18.0},
    '-O-':      {'Ecoh': 3350, 'Vw': 3.8},
    '-NH-':     {'Ecoh': 8370, 'Vw': 4.5},
    '-CONH-':   {'Ecoh': 33500, 'Vw': 9.5},
    '-SO2-':    {'Ecoh': 25560, 'Vw': 32.0},
    '-F':       {'Ecoh': 2240, 'Vw': 11.7},
    '-Cl':      {'Ecoh': 7960, 'Vw': 24.0},
}

def estimate_solubility_parameter(groups):
    """
    基于 Van Krevelen 基团贡献法估算溶解度参数
    groups: dict, 例如 {'-CH2-': 2, '-C6H4-': 1, '-COO-': 1}
    """
    total_ecoh = sum(GROUP_CONTRIBUTIONS[g]['Ecoh'] * n 
                     for g, n in groups.items())
    total_vw = sum(GROUP_CONTRIBUTIONS[g]['Vw'] * n 
                   for g, n in groups.items())
    
    if total_vw > 0:
        delta = np.sqrt(total_ecoh / (total_vw * 1e-6))
        ced = total_ecoh / total_vw
    else:
        delta = 0
        ced = 0
    
    return {
        'Ecoh_J_per_mol': total_ecoh,
        'Vw_cm3_per_mol': total_vw,
        'delta_MPa05': delta / 1000,
        'CED_J_per_cm3': ced,
    }

# 示例：PET (聚对苯二甲酸乙二醇酯)
pet_groups = {'-C6H4-': 1, '-COO-': 2, '-CH2-': 2}
result = estimate_solubility_parameter(pet_groups)
print(f"PET 溶解度参数: {result}")
