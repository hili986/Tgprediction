## 一、氢键影响 Tg 的机制与定量关系

### 1.1 基本机制

氢键通过以下机制提高聚合物 Tg：

1. **链段运动受限**：氢键形成物理交联点，限制链段协同运动
2. **有效交联密度增加**：氢键网络类似化学交联，增加约束数
3. **自由体积减少**：氢键使分子链堆积更紧密
4. **协同效应**：多重氢键产生非线性增强（如脲基双 N-H 供体）

### 1.2 定量数据：内聚能密度（CED）与 Tg 贡献

| 氢键基团 | CED (J/cm3) | 典型 Tg 提升 | 代表性聚合物 |
|---------|------------|-------------|------------|
| 脲基 (-NHCONH-) | ~2079 | 最高 | 聚脲 |
| 酰胺 (-CONH-) | ~1862 | 很高 (+50-80 K/mol%) | 尼龙系列 |
| 羟基 (-OH) | ~1500 | 高 | PVA, 纤维素 |
| 氨基甲酸酯 (-OCONH-) | ~1385 | 较高 | 聚氨酯 |
| 酰亚胺 (-CO-N-CO-) | ~980 | 中等 | 聚酰亚胺 |
| 磷酸酯 (P-O) | ~800-1200 | 中-低 | 聚磷酸酯 |
| 醚 (-O-) | ~400-600 | 低 | PEO |

**关键发现**：双氢键供体（如脲基 2x N-H）的 CED 约为单供体的 1.5 倍，但 Tg 提升可达 2-3 倍，体现协同效应。

### 1.3 Kwei 方程：氢键对共混体系 Tg 的贡献

    Tg = (w1 * Tg1 + k * w2 * Tg2) / (w1 + k * w2) + q * w1 * w2

- **q > 0**：组分间氢键强于纯组分内部 → Tg 正偏差
- **q = 0**：退化为 Fox 方程
- **q < 0**：混合削弱氢键 → Tg 负偏差

典型 q 值：强氢键 +20~+100K，中等 +5~+30K，弱/无 -10~+5K。

### 1.4 拓扑约束理论

Nakanishi et al. (2013) 框架：Tg ~ f(n_constraints / n_atoms)

每个氢键基团贡献拓扑约束，约束数/原子比值正相关 Tg。对核酸尤其有价值：碱基配对和堆积可用约束数量化。

### 1.5 水的增塑效应

| 聚合物 | 干态 Tg (°C) | 湿态 Tg (°C) | ΔTg (°C) |
|--------|-------------|-------------|----------|
| Nylon 66 | 80 | -25 ~ 0 | -80 ~ -105 |
| Nylon 6 | 60 | -10 ~ 10 | -50 ~ -70 |
| PVA | 85 | 30-40 | -45 ~ -55 |
| DNA 薄膜 | 170-180 | ~80-100 | -70 ~ -100 (估算) |

**对核酸的启示**：必须区分干态和湿态 Tg。核酸天然高度亲水，干态 Tg 更有参考价值。水含量可能需要作为独立特征。

## 二、七大桥梁聚合物家族

核酸（DNA/RNA）的关键氢键模式：
- **N-H...O=C**：碱基 Watson-Crick 配对（A-T/U 2 个, G-C 3 个）
- **N-H...N**：嘌呤-嘧啶配对
- **P-O 骨架**：磷酸二酯键（P=O 受体, P-O-H 供体）
- **O-H（核糖）**：RNA 2'-OH

以下聚合物家族共享这些氢键模式，可作为迁移学习的桥梁数据：

### 家族 1：聚酰胺 — N-H...O=C 模式

| 聚合物 | 结构 | Tg (°C) | 氢键密度 | 与核酸关联 |
|--------|------|---------|---------|-----------|
| Nylon 6 | -[NH-(CH2)5-CO]n- | 47-60 | 高 | 酰胺键模拟碱基配对 |
| Nylon 66 | -[NH-(CH2)6-NHCO-(CH2)4-CO]n- | 50-80 | 高 | 双酰胺更接近碱基对 |
| Nylon 46 | -[NH-(CH2)4-NHCO-(CH2)4-CO]n- | 80-90 | 很高 | 短间距高氢键密度 |
| Nylon 612 | -[NH-(CH2)6-NHCO-(CH2)10-CO]n- | 40-50 | 中等 | 长亚甲基稀释氢键 |
| Kevlar | 芳香酰胺 | 300+ | 极高 | 刚性骨架+强氢键 |

### 家族 2：聚氨酯 — N-H...O=C + O-H 模式

| 聚合物 | 硬段 Tg (°C) | 软段 Tg (°C) | 氢键特征 |
|--------|-------------|-------------|----------|
| MDI-BDO 基 PU | 100-120 | -40 ~ -60 | N-H...O=C 氨酯键 |
| TDI-BDO 基 PU | 80-100 | -40 ~ -60 | 不对称氨酯 |
| HDI 基 PU | 40-60 | -50 ~ -70 | 柔性链 |
| 聚脲 (PUa) | 120-200 | varies | 双 N-H，最强氢键 |

微相分离结构类似核酸的碱基堆积-磷酸骨架分离。

### 家族 3：聚酰亚胺 — 高 Tg + N-C=O 模式

| 聚合物 | Tg (°C) | 氢键特征 |
|--------|---------|----------|
| Kapton (PMDA-ODA) | 360-400 | C=O 受体，无 N-H 供体 |
| BPDA-PPD | 300-350 | 类似 |
| 6FDA-DAM | 380-400 | CF3 阻碍堆积 |
| PAI (聚酰胺-酰亚胺) | 250-300 | 兼有 N-H 供体和 C=O 受体 |

PAI 更完整地模拟碱基配对（同时含 N-H 供体和 C=O 受体）。

### 家族 4：聚磷腈 — P-N 骨架

| 聚合物 | 取代基 | Tg (°C) |
|--------|-------|---------|
| PDCP | -Cl | -66 |
| 苯氧基 | -OC6H5 | -8 ~ 0 |
| MEEP | -OCH2CH2OCH3 | -83 |
| 氨基 | -NHR | -20 ~ +35 |

P=N 骨架最接近核酸 P-O 骨架。取代基对 Tg 的影响范围 (~100K) 直接反映侧基氢键能力贡献。

### 家族 5：聚磷酸酯 — 直接匹配磷酸骨架

| 聚合物 | Tg (°C) | 特征 |
|--------|---------|------|
| 聚磷酸丙酯 | -30 ~ -10 | 最简单磷酸酯 |
| 聚磷酸乙二醇酯 | -40 ~ -20 | PEG 类柔性 |
| 聚磷酸酯（芳香侧基） | 20 ~ 60 | 芳香提升 Tg |
| 聚磷酰胺 | 0 ~ 40 | N-H + P=O |

与核酸磷酸二酯骨架化学等价，是最直接的桥梁聚合物。数据稀少。

### 家族 6：聚苯并咪唑 (PBI) — N-H...N 模式

| 聚合物 | Tg (°C) | 氢键特征 |
|--------|---------|----------|
| PBI | 427-435 | 强 N-H...N |
| AB-PBI | 400+ | 类似 |

N-H...N 氢键直接匹配核酸嘌呤-嘧啶碱基配对模式。

### 家族 7：核苷酸小分子 — 直接数据点

| 分子 | 分子量 | Tg (K) | Tg (°C) |
|------|-------|--------|---------|
| ATP | 507 | 246 | -27 |
| ADP | 427 | 244 | -29 |
| AMP | 347 | 249 | -24 |
| DNA 干膜 | 高分子量 | 443-453 | 170-180 |

ATP/ADP/AMP 的 Tg 非常接近（246-249K），磷酸基团数对小分子 Tg 影响不大。DNA 干膜 Tg 远高于核苷酸单体（~200K 跃升），归因于碱基堆积产生的刚性。

## 三、ML 特征工程：三层氢键描述符

### Tier 1：基础计数特征（RDKit 直接计算）

| 特征名 | RDKit 函数 | 描述 | 与核酸关联 |
|--------|-----------|------|-----------|
| NumHBD | CalcNumHBD() | 氢键供体数 | 碱基 N-H, 核糖 O-H |
| NumHBA | CalcNumHBA() | 氢键受体数 | C=O, N, P=O |
| NumLipinskiHBD | CalcNumLipinskiHBD() | Lipinski 标准 HBD | 更严格计数 |
| NumLipinskiHBA | CalcNumLipinskiHBA() | Lipinski 标准 HBA | 更严格计数 |
| TPSA | CalcTPSA() | 拓扑极性表面积 | 极性/亲水性 |
| HBD_ratio | NumHBD / NumHeavyAtoms | 供体密度 | 归一化 |
| HBA_ratio | NumHBA / NumHeavyAtoms | 受体密度 | 归一化 |

### Tier 2：氢键模式特征（SMARTS 匹配）

| 特征名 | SMARTS 模式 | 描述 | 与核酸匹配 |
|--------|------------|------|-----------|
| amide_count | `[NX3H][CX3](=O)` | 酰胺基团 | 碱基 C=O...H-N |
| urea_count | `[NX3H][CX3](=O)[NX3H]` | 脲基 | G-C 碱基对模式 |
| urethane_count | `[OX2][CX3](=O)[NX3H]` | 氨酯基 | 核苷键类似 |
| imide_count | `[CX3](=O)[NX3][CX3](=O)` | 酰亚胺基 | 嘌呤/嘧啶环 |
| phosphoester_count | `[PX4](=O)([OX2])([OX2])[OX2]` | 磷酸酯基 | 磷酸骨架直接匹配 |
| aromatic_NH | `[nH]` | 芳香 N-H | 碱基环 N-H |
| OH_count | `[OX2H]` | 羟基 | 核糖 2'-OH, 3'-OH |

### Tier 3：高级描述符

| 特征名 | 计算方法 | 复杂度 |
|--------|---------|--------|
| H_bond_energy_est | 基团加和法 | 中 |
| CED_contribution | Van Krevelen 基团贡献 | 中 |
| H_bond_network_density | 拓扑分析 | 高 |
| constraint_number | 拓扑约束理论 | 高 |
| stacking_propensity | 量化计算/ML | 很高 |

### 推荐最小特征集

```python
# Tier 1 必选
basic_features = [
    "NumHBD", "NumHBA", "TPSA",
    "HBD_ratio", "HBA_ratio",
    "NumRotatableBonds",  # 柔性指标
    "NumAromaticRings",   # 刚性/堆积指标
]

# Tier 2 推荐
hbond_pattern_features = [
    "amide_count", "urethane_count", "imide_count",
    "phosphoester_count", "aromatic_NH", "OH_count",
]

ml_features = basic_features + hbond_pattern_features
```

## 四、核酸 Tg 预测策略

### 4.1 核心挑战

1. **数据极度稀缺**：文献中 DNA/RNA Tg 数据不超过 10 个
2. **结构复杂性**：磷酸骨架 + 核糖 + 碱基三个层次
3. **氢键网络复杂**：Watson-Crick 配对 + 碱基堆积 + 骨架水合
4. **序列依赖性**：不同碱基序列产生不同的氢键网络

### 4.2 三层迁移学习策略

#### 策略 1：氢键模式迁移（推荐优先实施）

```
训练集：含核酸类氢键的合成聚合物（7 大家族）
    → 预训练模型：学习 "氢键模式 → Tg 贡献" 映射
    → 微调数据：少量核酸 Tg (ATP/ADP/AMP + DNA 干膜)
    → 目标：核酸序列 → Tg 预测
```

步骤：
1. 从 PolyMetriX 筛选 7 大家族聚合物 Tg 数据
2. 计算 Tier 1 + Tier 2 氢键特征
3. 训练 GBR/RF 基础模型
4. 用核酸数据微调

#### 策略 2：组分分解法（基于 Kwei 方程）

```
核酸 = 磷酸骨架 + 核糖 + 碱基
         ↓           ↓        ↓
    聚磷酸酯 Tg  PEG-类 Tg  嘧啶/嘌呤环 Tg
         ↓           ↓        ↓
    Kwei/Fox 方程混合 → 核酸 Tg 预测
```

需要：聚磷酸酯 Tg、核糖聚合物 Tg、碱基小分子 Tg、碱基配对的 q 参数。

#### 策略 3：分子基础模型微调

```
预训练模型 (TransPolymer/polyBERT/Uni-Mol)
    → 在聚合物 Tg 数据上 fine-tune
    → 在核酸相关数据上进一步 fine-tune
    → 核酸 SMILES → Tg 预测
```

优势：捕获隐式结构-性质关系。劣势：需更多核酸数据。

## 五、四层数据架构

```
Layer 4: 核酸 Tg 数据 (~10 条)            → 最终目标
Layer 3: 桥梁聚合物 Tg 数据 (~500-1000 条) → 迁移学习源
Layer 2: 通用聚合物 Tg 数据 (~13,000 条)   → 预训练基础
Layer 1: 小分子 Tg 数据 (~1000-3000 条)    → 辅助验证
```

### Layer 3 桥梁聚合物 SMARTS 筛选条件

```python
BRIDGE_POLYMER_FILTERS = {
    "polyamide": "[NX3H][CX3](=O)",                    # N-H...O=C
    "polyurethane": "[OX2][CX3](=O)[NX3H]",            # 氨酯键
    "polyurea": "[NX3H][CX3](=O)[NX3H]",               # 双 N-H
    "polyimide": "[CX3](=O)[NX3][CX3](=O)",            # 酰亚胺环
    "phosphoester": "[PX4](=O)",                        # P-O 骨架
    "benzimidazole": "c1ccc2[nH]cnc2c1",                # N-H...N
    "pyrimidine": "c1ccncc1",                           # 嘧啶环
    "purine": "c1cnc2[nH]cnc2n1",                       # 嘌呤环
    "hydroxyl_rich": "[OX2H]",                          # 核糖 O-H
}
```

### 推荐数据库 Schema

```sql
-- 主表
CREATE TABLE polymer_tg (
    id INTEGER PRIMARY KEY,
    name TEXT,
    smiles TEXT NOT NULL,
    bigsmiles TEXT,
    tg_value REAL NOT NULL,        -- Tg in Kelvin
    tg_method TEXT,                 -- DSC, DMA, etc.
    source TEXT,
    doi TEXT,
    layer INTEGER,                 -- 1=小分子, 2=通用, 3=桥梁, 4=核酸
    polymer_family TEXT,           -- polyamide, polyurethane, etc.
    -- 氢键特征（预计算）
    num_hbd INTEGER,
    num_hba INTEGER,
    tpsa REAL,
    hbd_ratio REAL,
    hba_ratio REAL,
    -- 氢键模式标注
    has_amide BOOLEAN,
    has_urethane BOOLEAN,
    has_urea BOOLEAN,
    has_imide BOOLEAN,
    has_phosphoester BOOLEAN,
    has_aromatic_nh BOOLEAN,
    has_hydroxyl BOOLEAN,
    -- 元数据
    molecular_weight REAL,
    is_dry_state BOOLEAN DEFAULT TRUE,
    water_content REAL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 核酸专用扩展表
CREATE TABLE nucleic_acid_tg (
    id INTEGER PRIMARY KEY,
    polymer_tg_id INTEGER REFERENCES polymer_tg(id),
    sequence TEXT,                  -- 碱基序列 (ATCG...)
    sequence_length INTEGER,
    gc_content REAL,
    base_composition TEXT,         -- JSON: {"A": 0.3, "T": 0.2, ...}
    is_double_stranded BOOLEAN,
    backbone_type TEXT,            -- DNA / RNA / PNA / LNA
    modification TEXT
);
```

## 六、行动计划

### 近期 (1-2 周)

1. **数据获取**
   - 下载 figotj/Copolymer 数据集 (含共聚物 Tg)
   - 收集核苷酸 Tg 文献数据 (ATP/ADP/AMP/DNA 干膜)

2. **SMARTS 筛选脚本**
   - 输入 SMILES 列表 → 输出桥梁聚合物筛选结果
   - 集成 RDKit 特征计算 (Tier 1 + Tier 2)

3. **数据库搭建**
   - 创建 SQLite 数据库（按上述 Schema）
   - 导入 Bicerano + PolyMetriX 数据

### 中期 (3-6 周)

4. **特征工程与模型训练**
   - 在现有 GBR 模型上添加氢键特征
   - 对比有/无氢键特征的性能
   - 特征重要性分析

5. **桥梁聚合物模型**
   - 用 Layer 3 数据训练专用模型
   - 准备迁移学习框架

### 长期 (7-12 周)

6. **核酸 Tg 预测**
   - 实施迁移学习策略 1
   - 核酸序列特征编码
   - 模型验证与优化
