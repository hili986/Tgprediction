## 1. 什么是玻璃化转变温度 (Tg)

**玻璃化转变温度 (Glass Transition Temperature, Tg)** 是非晶态高分子从硬而脆的**玻璃态**转变为柔软有弹性的**高弹态**的温度。它是高分子材料最重要的热力学性质之一。

用生活中的类比来理解：
- **低于 Tg**：聚合物像玻璃一样硬脆。冬天室外的塑料管踩上去会碎裂。
- **高于 Tg**：聚合物变得柔软、有弹性。橡皮筋在室温下是柔软的，因为天然橡胶的 Tg 约为 -70 C，远低于室温。

```
温度升高 -->
+--------------+------------------+------------------+
|  玻璃态       |  玻璃化转变区     |  高弹态           |
|  链段冻结     |  链段开始运动     |  链段自由运动     |
|  仅振动/摆动  |  协同运动解冻     |  长程构象变化     |
|  硬、脆       |  粘弹性          |  柔软、有弹性     |
|              |  <-- Tg 在此 --> |                  |
+--------------+------------------+------------------+
```

**关键概念**：Tg 不是一个精确的点，而是一个温度区间（通常 10-30 C 宽）；Tg 取决于测量速率（加热越快，测到的 Tg 越高）；Tg 是动力学现象，不是热力学相变。

### 常见聚合物 Tg 一览

| 聚合物 | 简称 | Tg (C) | 室温下状态 | 日常用途 |
|--------|------|--------|-----------|---------|
| 聚二甲基硅氧烷 | PDMS | -125 | 液态/弹性体 | 硅胶、润滑脂 |
| 聚乙烯 | PE | -78 | 柔软塑料 | 保鲜膜、塑料袋 |
| 天然橡胶 | NR | -70 | 弹性体 | 轮胎、手套 |
| 聚丙烯 | PP | -10 | 半刚性塑料 | 食品容器 |
| 尼龙 6 | PA6 | 47 | 工程塑料 | 齿轮、纤维 |
| 聚苯乙烯 | PS | 100 | 刚性塑料 | 泡沫饭盒、模型 |
| 聚碳酸酯 | PC | 150 | 透明刚性塑料 | 眼镜片、光盘 |
| 聚酰亚胺 (Kapton) | PI | 360 | 耐高温薄膜 | 航天、电子 |

### 影响 Tg 的核心因素

| 因素 | 影响方向 | 物理解释 | 例子 |
|------|---------|---------|------|
| 主链刚性 | Tg 升高 | 刚性主链更难运动 | PI (360 C) vs PE (-78 C) |
| 侧基体积 | Tg 升高 | 大侧基阻碍链段运动 | PS (100 C) vs PE (-78 C) |
| 氢键 | Tg 显著升高 | 分子间额外约束 | PA6 (47 C) vs PE (-78 C) |
| 交联密度 | Tg 升高 | 化学键限制运动 | 硫化 vs 未硫化橡胶 |
| 增塑剂 | Tg 降低 | 小分子增大自由体积 | 增塑 PVC vs 硬质 PVC |

**核心规律**：凡是限制链段运动的因素都升高 Tg，凡是增加自由体积或增强链段活动性的因素都降低 Tg。

### Tg 的测量方法

实验中测量 Tg 的主要方法：

| 方法 | 原理 | 精度 | 耗时 |
|------|------|------|------|
| **DSC**（差示扫描量热法） | 测量比热容的突变 | +-2-5 C | 1-2 小时 |
| **DMA**（动态力学分析） | 测量模量和损耗角 | +-1-3 C | 2-4 小时 |
| 介电弛豫谱 | 测量介电常数变化 | +-2-5 C | 数小时 |
| 膨胀计法 | 测量体积变化 | +-5-10 C | 数小时 |

**注意**：不同方法、不同加热速率测出的 Tg 可能相差 5-15 C。这就是为什么数据库中同一种聚合物可能有不同的 Tg 值——这也给数据驱动的预测带来了额外噪声。

### Tg 预测方法的演进

```
1950s-1990s          2000s-2015           2016-2022            2023-2026
基团贡献法           QSPR + ML            深度学习兴起          大模型 + 多保真度
Van Krevelen         RDKit + RF/SVR       GNN, Transformer     预训练微调
Bicerano             Morgan 指纹          polyBERT             虚拟数据增强
手工查表             自动描述符            TransPolymer         TabPFN 零调参
R2~0.85-0.90         R2~0.88-0.92         R2~0.90-0.93         R2~0.95-0.97
```

当前最先进方法的关键发现：**特征工程质量 > 数据规模 > 模型复杂度**。Zhao et al. (2025) 仅用 4 个精选物理特征在 ~700 样本上达到 R2=0.97，超过了所有 GNN 和 Transformer 方法。

## 2. 为什么要预测 Tg

### 工业价值

1. **决定使用温度范围**：产品在 Tg 附近会突然变软失去强度，工程设计必须知道 Tg 才能选材。
2. **决定加工温度**：注塑成型、挤出等加工必须在 Tg 以上进行。Tg 太高则加工困难，太低则产品太软。
3. **药物递送**：核酸-高分子载体的 Tg 影响药物释放速率和稳定性。

### 学术价值

4. **材料发现加速器**：实验测一个 Tg（DSC/DMA）需要先合成样品（天-周级别），AI 预测能在合成前筛选候选材料。
5. **高分子信息学 Benchmark**：Tg 是聚合物信息学中研究最广泛的基准性质。
6. **理论未解之谜**：2005 年《Science》125 个未解问题之一："玻璃态的本质是什么？"。目前没有任何理论能从聚合物结构 ab initio 精确预测 Tg。

正是因为理论无法直接计算 Tg，我们转向**数据驱动的 AI 方法**——用机器学习从实验数据中学习"结构到Tg"的映射关系。

## 4. 核心挑战

### 挑战一：数据稀缺

高质量聚合物 Tg 实验数据有限。Bicerano 经典数据集仅 304 条均聚物；核酸 Tg 数据全球不超过 10 个数据点。不同来源（DSC vs DMA）、不同加热速率测出的 Tg 可能相差 5-15 K。

### 挑战二：理论无解

Tg 是动力学现象，涉及极慢的弛豫过程和成千上万个链段的协同运动。分子动力学模拟的冷却速率比实验快约 10 个数量级，导致模拟 Tg 偏高 50-100 K。

### 挑战三：结构多样性

高分子结构从简单线性均聚物到复杂的嵌段、接枝、交联、超支化结构，跨度极大。单一表示方法和模型难以覆盖所有类型。

### 挑战四：核酸迁移困难

核酸的 Tg 与合成聚合物的 Tg 在物理本质上不同——合成聚合物 Tg 反映链段运动冻结，而核酸热转变主要由碱基堆积、配对氢键和水合层驱动。用合成聚合物模型预测核酸 Tg 本质上是跨物理概念的迁移。

### 挑战五：单体描述符天花板

当前主流方法从重复单元 SMILES 计算描述符，丢失了链级效应（分子量、立构规整性）和体相效应（结晶度、交联密度）。理论上限 R2 约 0.84，需要更高层次的表示方法（VPD、GNN）来突破。
