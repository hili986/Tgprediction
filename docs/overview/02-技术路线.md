## 1. 方案 A：物理特征增强路线（核心，已完成）

### 1.1 核心思路

用**物理直觉引导的特征工程**替代/补充自动生成的描述符，让 ML 模型获得更多物理信息。核心突破是 VPD（虚拟聚合描述符）——通过模拟 3 个重复单元的寡聚体来捕获聚合效应，突破单体描述符的理论天花板。

### 1.2 多层级特征体系 M2M-V (46 维)

```
M2M-V 特征体系 (46 维)
  |
  +-- L0 Afsordeh (4 维)     # 物理直觉特征
  |     FlexibilityIndex      # 主链柔性 (SHAP #1)
  |     SOL                   # 溶解度参数估计
  |     HBondDensity          # 氢键密度
  |     PolarityIndex         # 极性指数
  |
  +-- L1 RDKit 2D (15 维)    # 化学信息学描述符
  |     RingCount, NumRotatableBonds, TPSA, FractionCSP3,
  |     NumHDonors, NumHAcceptors, MolLogP, BalabanJ, ...
  |
  +-- H-bond SMARTS (15 维)  # 氢键模式特征
  |     amide, urea, urethane, imide, phosphoester, ...
  |     + CED 加权和          # 内聚能密度估计
  |
  +-- VPD (12 维)            # 虚拟聚合描述符 (核心创新)
        3-RU 寡聚体 vs 单体的差异描述符
        RingCount_per_RU, junction_flex_ratio,
        TPSA_per_RU, RotBonds_delta, LogP_delta, ...
```

**VPD 的物理意义**：将重复单元 SMILES "虚拟聚合"为 3 个单元的寡聚体，计算聚合前后描述符的变化。这些差异直接反映了聚合带来的链级效应（连接点刚性变化、极性变化等），是突破单体描述符天花板的关键。

### 1.3 评估方法：Nested CV

```
外层: RepeatedKFold(n_splits=5, n_repeats=3)  # 15 个评估折
  |
  +-- 内层: KFold(n_splits=3)                   # 超参优化
        |
        +-- 预处理: PowerTransformer(Yeo-Johnson) + MinMaxScaler
        +-- 模型训练 + 超参搜索
  |
  +-- 外层测试集评估 (R2, MAE, RMSE)
```

Nested CV 保证评估无偏——超参选择不会泄露到测试集评估中。

### 1.4 五阶段实验设计

| 阶段 | 内容 | 数据 | 核心产出 |
|------|------|------|---------|
| Phase 1-3 | 基础搭建 | Bicerano 304 | 特征体系 + 评估框架 + 桥梁迁移 |
| Phase 4 | 消融实验 (E1-E15) | Bicerano 304 | VPD +4.6%, GNN 端到端失败 |
| Phase 5 | 模型对比 (E16-E24) | 统一 7,486 | TabPFN SOTA, 特征选择, UQ |
| Phase 5B | 收尾 (E25-E26) | 统一 + 桥梁 | 核酸预测, Optuna 调参 |

## 2. 方案 B：GNN 混合路线（待启动，需 GPU）

### 2.1 核心思路

用 GNN（图注意力网络）从分子图中自动提取特征，与手工描述符拼接，送入传统 ML 模型。参考 GATBoost (Park et al., 2024, R2=0.91) 的混合架构。

### 2.2 Tandem 架构

```
输入 SMILES
  |
  +-- 路径 A: 分子图 --> Physics-GAT --> 图嵌入 (64 维)
  |                       (3 层 GAT, 4 heads)
  |
  +-- 路径 B: SMILES --> M2M-V 特征管线 --> 表格特征 (46 维)
  |
  +-- 拼接: [图嵌入 || 表格特征] = 110 维
  |
  +-- GBR / CatBoost --> Tg 预测
```

### 2.3 预训练策略

```
Stage 1: 自监督预训练
    数据: 46K Fox 虚拟共聚物 + 22K 外部数据
    任务: 掩码原子预测 + 对比学习
    目标: 学习通用分子表示

Stage 2: 有监督微调
    数据: 7,486 统一数据集
    任务: Tg 回归
    方法: 冻结 GNN 底层, 微调顶层 + 回归头

Stage 3: 混合评估
    提取 GNN 嵌入 + M2M-V 特征 --> 传统 ML
    消融: 纯 GNN vs 纯描述符 vs 混合
```

### 2.4 Phase 4 已有验证

在 Bicerano 304 条数据上的预实验结果：

| 配置 | R2 | 结论 |
|------|-----|------|
| GNN 端到端 | <0 | 304 条完全无法训练 GNN (参数/样本比 >300:1) |
| GNN 嵌入 + GBR | 0.839 | 可行但未超越 VPD (0.866) |

方案 B 的核心假设：大规模预训练（46K+22K 数据）能否突破当前 GNN 嵌入的上限？需要 A800 GPU 验证。

## 3. 关键决策记录

### 3.1 为什么方案 A 先做

| 考量维度 | 方案 A | 方案 B |
|---------|--------|--------|
| 技术栈兼容 | 100% (纯 scikit-learn) | 需新增 PyTorch + PyG |
| GPU 需求 | 无 | 必须 |
| 实现难度 | 低-中 | 中-高 |
| 预期 R2 上限 | 0.90-0.97 | 0.88-0.92 |
| 可解释性 | 强 (每个特征有物理意义) | 弱 (图嵌入黑箱) |
| 时间投入 | 2-3 周 | 3-4 周 |

**结论**：方案 A 投入产出比最高，技术栈零变化，且 Zhao et al. 证明精选物理特征可达 R2=0.97。

### 3.2 单体描述符天花板与 VPD 突破

**天花板问题**：所有特征从重复单元 SMILES 计算，等价于用小分子近似高分子链。丢失了链级效应（分子量、立构规整性）和体相效应（结晶度、交联密度）。理论上限 R2 约 0.84。

```
聚合物 Tg = f(单体结构, 链级效应, 体相效应)
                |              |            |
           我们能捕获的   VPD 部分捕获   完全丢失的
           (~70% 变异)   (+4.6% 提升)   (~10% 变异)
```

**VPD 如何突破**：通过构建 3-RU 寡聚体，计算聚合前后描述符差异，部分捕获了链级效应中的"连接效应"——这是方案 A 最核心的创新。

### 3.3 已锁定的技术选择

| 决策点 | 选择 | 理由 |
|--------|------|------|
| 主模型 | ExtraTrees / TabPFN | SOTA for small-medium datasets |
| 评估方法 | Nested CV (outer=5x3, inner=3) | 无偏评估，文献标准 (Xu 2024) |
| 预处理 | PowerTransformer + MinMaxScaler | Yeo-Johnson 处理偏态分布 |
| 特征选择 | VarianceThreshold -> Boruta -> mRMR -> SHAP | 四阶段渐进筛选 |
| 集成策略 | Stacking (CatBoost+LightGBM+ET+XGBoost -> Ridge) | 7K+ 数据才有效 |
| 多保真度 | 两阶段训练（预训练->微调） | 用户指定 |
| 迁移策略 | 特征迁移（加权训练） | 用户指定 |
| 调参 | Optuna 自动搜索 20 trials/fold | 自动化，可复现 |

### 3.4 方案选择中排除的方案

| 方案 | 预期 R2 | 排除原因 |
|------|---------|---------|
| C: Transformer (polyBERT) | 0.88-0.93 | 黑箱性强，768 维嵌入不可解释 |
| D: 纯数据驱动迁移 | 0.87-0.90 | 不解决描述符天花板根本问题 |
| E: 全栈多模态 (MMPolymer) | 0.90-0.94 | 实现复杂度极高，超出 SITP 时间预算 |

## 4. 改进方向评估（6 个方向）

在方案选择前，系统评估了 6 个改进方向：

| 方向 | 核心方法 | 预期 R2 | 与现有栈兼容 | 实际采用 |
|------|---------|---------|-------------|---------|
| 1. GNN | 图注意力网络自动学习 | 0.88-0.91 | 需 PyTorch | 方案 B |
| 2. Transformer | polyBERT 预训练嵌入 | 0.88-0.93 | 需 PyTorch | 未采用 |
| 3. 多模态 | 1D SMILES + 3D 结构 | 0.90-0.94 | 全新栈 | 未采用 |
| **4. 物理引导** | **精选物理特征 + 残差** | **0.88-0.97** | **完全兼容** | **方案 A 核心** |
| 5. 迁移学习 | 自监督预训练 + 数据增强 | 0.87-0.90 | 兼容 | 部分采用 |
| 6. 核酸专用 | 目标重定义 + 桥梁迁移 | N/A | 兼容 | 部分采用 |

**关键洞察**：Zhao et al. 用 4 个精选物理描述符 + ExtraTrees 达到 R2=0.97，超过所有 GNN 和 Transformer 方法。这证明**特征工程质量 > 模型复杂度**。
